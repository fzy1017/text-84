☞复合选择器：

    1. 后代选择器：后代选择器只能用在嵌套结构中，并且用来选中的是后代元素。
                  【所有后代元素，所有的子元素】

        语法：

             选择器 选择器 {
                 属性：值;
             }

        注意事项：
              1. 选择器和选择器之前必须设置 空格

              2. 在后代选择器中，前面的选择器代表父元素，最后一个选择器代表的是要被选中的后代元素（子元素）


    2. 子代选择器：子代选择器只能用在嵌套结构中，并且用来选中的是子代元素，选中的是直接子元素
                （不包括间接子元素）


        语法：

            选择器>选择器 {
                属性： 值；
            }

        注意事项：   

            1. 选择器和选择器之间必须写 >

            2. 第一个选择器选中的是父元素，第二个选择器选中的是父元素中的直接子元素。

    3. 标签指定式选择器（交集选择器）：
        
         注意事项：
             1. 标签指定式选择器满足  '既。。。。又。。。'的关系。

    4. 并集选择器：

         语法：
             选择器,选择器 {
                 属性： 值;
             }

          总结：
             1. 如果希望给一些标签设置相同的样式，那么可以使用类选择器，还可以使用并集选择器

             2. 并集选择器与HTML结构无关

☞HTMl标签中元素的分类：按照标签在网页的显示方式进行的分类（为了网页布局）。


    ✔ 块级元素:  

          代表元素：  div,p, h1,h2,h3, ul,ol,dl,li,dd....

          特点：
             1. 块级元素独占一行显示

             2. 块级元素默认如果没有设置宽度，那么其宽度默认等于他父元素的宽度

             3. 块级元素可以通过设置宽度或高度改变大小


           div标签：
               1. 没有任何语义的一个标签

               2. 在网页布局过程中需要使用div进行布局。
               
    ✔ 行内元素：
          代表元素： span, a, font，strong, em， i.....

          特点：
             1. 所有的行内元素都在一行显示。

             2. 行内元素宽度或高度，都是被其内容撑开。

             3. 行内元素不能设置宽度和高度。

         span标签：

             1. 没有任何语义的标签。

             2. 在网页布局中，如果希望元素一行显示，可以使用行内元素span。
              
    ✔ 行内块元素

          代表元素： img, input

          特点：
              1.  所有的行内块元素都在一行显示。
              2.  行内块元素可以设置宽度和高度。

☞ CSS三大特性：

     ✔ 层叠性：

         使用同一种选择器选中的是同一个标签，最后定义的样式会将前面定义的样式覆盖掉（重复的样式）


     ✔ 优先级：
         
        使用不同选择器选中同一个标签，那么就要按照优先级从高到低显示样式
         

       1.... 标签选择器 < 类选择器  < ID选择器  < 行内样式  < !important

            10        100        1000       10000       100000


        特点：
            1. 继承的权重为0: 在嵌套结构中，如果父元素和子元素同时有样式，那么最后会按照子元素的样式显示


            2. 权重可以叠加运算：如果使用不同选择器选中同一个标签，那么可以考虑权重可以叠加


    ✔ 继承性：在嵌套结构中，子元素(不能有自己的样式)会受父元素样式的影响.

         1. 前景色color可以被继承

         2. font中的所有属性都可以被继承。
