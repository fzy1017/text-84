☞ 今日目标： 浮动（float）

     1. 伪元素： 通过CSS代码模拟了一个标签（块级元素，行内元素，行内块元素）

         ✔  ::before {}

         ✔  ::after {}

         总结：
             ◆ 在用法上我们就可以把伪元素当做一个标签使用

             ◆ 伪元素必须设置content属性

             ◆ before 始终在结构的最开始位置    after 在html结构中结束位置

             ◆ 伪元素是行内元素

    2. 浮动

         ◆为什么要学习浮动？

             ☞ 在网页布局中，当我们希望块级元素在一行显示的时候就必须使用浮动（网页布局）
             
         ◆浮动在网页中的作用？

             ☞ 可以实现让块级元素在一行显示。

         ◆为什么元素设置完浮动后，会在一行上显示？

             标准流：在网页中元素的默认显示方式

             因为： 浮动的元素脱离的标准流的显示方式(脱标)


          ◆ 浮动的特点：

               ☞ 浮动可以让元素不占位置（脱离标准流）

               ☞ 如果要实现元素一行显示，那么需要给元素都设置浮动。


          ◆浮动使用总结：

                1. 当块级元素一行显示的时候就使用浮动

                2. 通过浮动实现一行显示的元素，必须都是设置浮动。


☞ 清除浮动： 消除浮动对元素的影响

       1. 使用clear属性清除浮动

         clear: left | right  | both;


         步骤：
            ✔ 找到浮动的元素，然后再浮动元素的后面添加一个空的div

            ✔ 选中该div，然后在div中设置  clear : both;


        2. 使用伪元素清除浮动

          /* 第二种方式: 伪元素清除浮动 */
         .clearfix::after {
             content: "";
             display: block;
             clear: both;

             /* 补充代码 */
             height: 0;
             line-height: 0;
             /* 将该标签隐藏 */
             visibility: hidden;
         }

         .clearfix {
              /* 兼容IE浏览器 */
              *zoom: 1;
         }

         3.可以给父元素设置overflow:hidden;

☞清除浮动的总结：

      1. 不是所有的浮动都需要清除浮动。

      2. 如果父元素没有设置高度，且父元素中所有的子元素都设置了浮动（清除浮动）

      3. 元素可以设置高度也可以不用设置高度

      4. 推荐使用伪元素清除浮动
