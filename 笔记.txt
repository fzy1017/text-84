2.CSS的语法结构：

    ☞选择器 {

        属性: 值;
    }

        ☞选择器： CSS用来选中标签的一种方式

        ☞属性 ：就是HTML标签中对应的具体样式

        ☞总结几个常见的属性：

            color属性： 在css中也是用来改变文字颜色 【前景色】

            background-color属性： 在CSS中设置背景色

            font-size属性: 在CSS中是用来设置文字大小

            width属性： 在CSS中用来设置标签的宽度

            height属性： 在CSS中用来设置标签的高度

            text-align属性： 设置内容（文字，图片）对齐方式的

                   left | center | right

            text-indent: 设置文字缩进


    ☞选择器分类介绍：

        ✔ 简单选择器：
             
            1. 标签选择器（入门）：通过HTMl标签名选中对应的标签

                   HTMl标签名 {
                       属性: 值;
                       ...
                   }

                   特点：
                       ✔ 可以将页面中所有符合html标签名的标签选中，与html标签的结构无关

             2. 类选择器（必须掌握）： 同过类名选中对应的标签

                  语法：

                  1 .自定义类名 {
                      属性：值;
                    }

                  2. 标签必须设置class属性，class属性中的值 = 自定义类名 才可以


                  总结：
                     ✔. 类样式其实就是一个公共的样式，任何标签都可以通过class属性使用

                     ✔. 同一个标签可以同时通过class调用多个类样式

                     例如： <p class="abc  big">段落标签</p>


                   3. 自定义类名的规范问题：   

                        ✔ 类名不能使用纯数字或者以数字开头

                        ✔ 类名不能使用纯特殊符号或者以特殊符号开始

                        ✔ 不推荐使用汉字定义类名

                        ✔ 建议使用具有语义的单词或者拼音定义类名
                        

             3. ID选择器（了解）：通过标签ID属性选中标签的。

                 语法：

                    #自定义的ID {

                        属性: 值;

                    }

                   总结：
                       ✔ ID选择器与类选择器在用法上基本一样

                       ✔ 类选择器在使用过程中是通过class属性使用的

                       ✔ ID选择器在使用过程中是通过ID属性使用的。

                       ✔ 不建议多个标签使用同一个ID样式（ID要保证唯一性）

                       ✔ 一个标签一次只能调用一个ID样式
                   

             4. 通配符选择器（了解）

                   * {
                       属性： 值;
                   }

                 总结：
                     1. 通配符选择器是将页面中所有的标签都选中了

                     2. 当页面中所有的标签具有相同的样式的时候，可以使用通配符选择器更简单。
              

        ✔ 复合选择器：其实就是简单选择器组合使用

             1. 后代选择器（必须掌握）

             2. 子代选择器（了解）

             3. 标签指定式选择器（掌握）

             4. 并集选择器（掌握）

3. CSS书写位置介绍：

      ✔ 内嵌式写法： 将CSS代码写到HTMl页面中。（课堂主要的写法）
          
          1. 在html页面中的head标签中添加一个style标签

          2. 在style标签中写对应的CSS代码

          例如
           <style>
             CSS代码
           </style>

      ✔ 外联式写法：将CSS代码单独放到一个CSS文件中（独立项目的时候推荐使用）

            1. 先创建一个CSS文件，后缀名 .css

            2. 在CSS文件中写对应的CSS代码

            3. 在html页面中通过link标签将CSS文件引入到当前html页面中

            例如：

            <link rel="stylesheet" href="test.css">

      ✔ 行内式写法： 将CSS代码直接写到标签内部（很少使用）

            1. 在标签中添加一个属性  style属性即可

            例如：
             <p style="CSS代码">文字</p>


4. 其他部分介绍：

       ✔ 与文字相关的属性：

            1. 文字大小:  font-size

            2.  文字字体:   font-family:

                 注意：  
                    ◆ 设置文字字体，可以使用中文

                    ◆ 设置文字字体，可以使用英文的表示方式

                    ◆ 设置文字字体, 可以使用uncode编码表示

            3. 设置文字加粗显示：  font-weight: bold(700) 加粗 |  normal(400) 不加粗

            4. 设置文字斜体显示：  font-style: italic(斜体) | normal (正常显示)

            5. 设置文字的行高： 在设置文字的上下间距( line-height)


      ✔ 文字属性合写方式：
            
             font: font-style   font-weight  font-size /   line-height  font-family;


             注意事项：
                  1. 必须设置 font-size 和 font-family

                  2. font-size 必须在font-family之前设置

                  3. 如果要设置line-height 那么必须把line-height放到 font-siez 之后，并且以 "/" 隔开

                  4. 不能再属性合写中设置文字颜色。


      ✔ 颜色表示方式（了解）：
            
             1. 系统预定义颜色：  red , blue , grenn .....

             2. 使用十六进制表示颜色：

                    #0-9A-F （一共是6位）


             3. 使用rgb表示颜色

                     r  red

                     g  green

                     b  blue


                     注意

                         每一个颜色的取值范围都是   0 - 255


            4. 实现颜色半透明效果：

                     ✔ 使用rgba()

                        a  alpha(通道，透明度)     取值： 0-1


                     例如：  background-color: rgba(9,9,86, 0.5);



                     ✔ 使用opacity设置半透明效果  取值： 0-1
